A fact-checking system that verifies user claims with evidence-backed explanations using web searches and AI, delivering clear, transparent verdicts through a simple interface.

### Features
- Automated claim verification
- Evidence-backed fact-checking
- Web search integration for real-time data
- AI-driven analysis and explanations
- Transparent verdict delivery

### Technologies Used
- Python
- TensorFlow
- Natural Language Inference
- Google custom Search
- RAG
